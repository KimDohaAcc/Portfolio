📖 프로젝트 소개

부동산 전월세 실거래가 확인 플랫폼으로 원하는 지역을 검색하여 해당 지역의 전월세 실거래가를 볼 수 있습니다.


지역 검색: 지도를 통해 원하는 지역을 선택하여 부동산 매물에 대한 정보를 확인할 수 있습니다.


전세 실거래가: 그래프를 통해 전세 가격의 변동을 시간에 따라 확인할 수 있습니다.


월세 실거래가: 월세 거래 내역을 상세히 살펴볼 수 있습니다.


댓글 작성: 각 매물에 대해 댓글을 작성하여 다른 사용자들과 소통할 수 있습니다.


관심 매물 등록: 관심 있는 매물을 등록하여 추후에 쉽게 찾아볼 수 있습니다.


부동산 선호 설정: 회원가입 시 사용자는 부동산 선호사항을 설정할 수 있습니다. 이를 토대로 선호 기반 매물을 검색할 수 있습니다.

회원의 선호 기반 매물 추천


보안, 회원 관련 기능 구현



🎯 기술 리뷰


[Spring security 적용]


Spring security 2.7 버전을 사용하여 비밀번호를 암호화 하였으며, 회원과 관리자의 역할을 나눠 관리자 페이지의 접근을 막았습니다. 그 과정에서 필요한 config 파일 작성 방법과 보안 처리를 익힐 수 있었으며, jsp에서 spring security tag library를 활용하여 간편하게 권한 확인을 할 수 있었습니다.


[pageable을 활용한 pagination 처리]


Pageable을 이용하여 페이지네이션 처리를 구현하였습니다. 전월세 데이터가 많이 존재하는 경우 페이지네이션으로 데이터를 효율적으로 관리하고 사용자에게 보다 편리한 서비스를 제공할 수 있었습니다.
