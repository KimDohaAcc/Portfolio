📖 프로젝트 소개

임산부의 90%에게서 발생하는 산후우울감 및 산후우울증을 관리할 수 있는 임산부 중심의 SNS 서비스입니다.


자신의 감정을 되돌아볼 수 있는 감정 일기를 제공하며, 기간별 감정 그래프 및 통계를 볼 수 있습니다.


같은 공감대를 가진 임산부의 게시물을 열람할 수 있으며 팔로워와 채팅할 수 있습니다.


비대면 진료 서비스를 제공해 거동이 불편한 임산부를 고려했습니다.


수유실 정보 API를 활용한 수유실 지도가 있습니다.

🎯 기술 리뷰


[그룹채팅 기능 추가]


1:1 채팅 기능을 개발하던 중간에 그룹채팅 기능을 추가하게 되었습니다. 이를 위해 채팅방 엔티티를 만들고 1:1 채팅 및 그룹 채팅에서 활용할 수 있게 하였습니다. 또한 토픽과 메시지 리스너 코드를 추가했습니다. 추후 유지보수를 하며 Kafka 컨슈머를 1:1 채팅용과 그룹채팅용으로 분리하여 용도에 맞게 컨슈머의 batch를 튜닝했습니다. 이를 통해 1:1 채팅은 메시지를 빠르게 처리해 신속성을 높였으며, 그룹 채팅은 큰 규모로 처리해 안정성을 높였습니다.


[Gerrit을 활용한 코드 리뷰]


Google에서 개발한 코드 리뷰 툴인 Gerrit을 활용하여 프로젝트의 코드 리뷰를 진행하였습니다. Github의 Pull Request나 Gitlab의 Merge Request만 활용하여 코드 리뷰를 해오다가 처음으로 코드 리뷰 툴을 활용해볼 수 있었습니다. 커밋 단위로 신속하게 코드 리뷰를 하며 피드백을 받아볼 수 있었으며, 리뷰를 하지 않으면 레포지토리에 올라가지 않아 모든 팀원들이 코드 리뷰에 적극적으로 참여하는 경험을 할 수 있었습니다.
