📖 프로젝트 소개

아이중심적인 대다수의 임산부 어플과 달리 산모중심의 SNS 어플을 개발하고자 하였습니다.


주요 기능

  • 산모 / 육아일기를 작성할 수 있는 다이어리
  • 피드, 채팅 좋아요, 댓글, 팔로우 기능을 제공하는 SNS
  • 진료 예약, 비대면 화상 진료, 처방전 발급

UI


🙋‍♂️ 담당 역할

DirectMessage, GroupChat, 화상 통화, 피드, 댓글, 좋아요, 해시태그


🎯 기술 리뷰

[Kafka 및 MongoDB를 활용한 실시간 DirectMessage 구현]

Kafka를 사용하여 메시지 큐 및 이벤트 스트리밍을 구현하고, MongoDB를 데이터 저장 및 관리에 활용하여 안정적이고 확장 가능한 채팅 애플리케이션을 개발했습니다. 또한 STOMP를 사용하여 WebSocket을 통해 실시간 통신을 구현하여 사용자 간의 채팅을 가능하게 했습니다.

프로젝트 중반에 그룹 채팅 기능을 만들게 되었습니다. 이미 Kafka 설정을 DM에 맞게 해둔 터라 설정 및 리스너를 변경해야 했습니다. 팀원들과 상의하여 DM과 그룹 채팅의 컨슈머를 분리하고 토픽을 새로 구성하여 성공적으로 그룹 채팅 기능까지 구현할 수 있었습니다.
